make sure you have working code and it is visible on the viewport

.ARTX

If you choose this you will get an .artx file with your code in it, which you or someone else can open in the artx editor with the import button and it can then be edited again.

.PNG

if you choose this you will get a png photo, this is enlarged by 50x because otherwise the quality is too poor, this is just a normal png file with nothing special

.SVG

If you choose this you will receive an SVG file. You can use this file to edit in Illustrator or just use it because it immediately works perfectly as you want it to, it can be zoomed in forever and it takes up little memory.
