The Lab

The lab wordt vanaf nu gebruikt als een plek om al mijn ontdekkingen en ervaringen te delen. Hier gaan stukken code staan die ik heb geschreven en die ik interessant vind, dingen in Photoshop, Illustrator, Blender, Cinema 4D, of welk programma dan ook. Dat komt allemaal hierop.

Ik heb achter de schermen een programmaatje gemaakt waarmee ik deze post kan maken. Dit systeem werkt door een aantal requests te maken naar de GitHub API, om precies te zijn 3. De eerste request is er om de data die ik in mijn programmaatje heb, naar GitHub te sturen. Dit maakt een bestand aan in het "thelab" mapje. Het bestand heet hetzelfde als het onderwerp. De tweede request is erom te kijken wat er in het lab.html bestand zit, en dan doet de derde request er de foto, datum en nummer in die bij het onderwerp hoort. Dit gaat allemaal automatisch.
